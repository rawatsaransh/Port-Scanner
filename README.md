# 🔍 PortPulse - Python Port Scanner

PortPulse is a fast, multi-threaded TCP port scanner built using Python.  
It is designed for educational and ethical cybersecurity learning purposes.

> ⚠️ Disclaimer: This tool is strictly for educational use only.  
> Always scan systems/networks that you own or have explicit permission to test.

---

## 🚀 Features

- ⚡ Fast multi-threaded scanning using ThreadPoolExecutor
- 🎯 Supports domain names and IP addresses
- 🔢 Custom port range selection
- ⏱ Socket timeout handling for efficiency
- 🧾 Real-time display of open ports
- 🛡 Input validation and error handling
- 📊 Shows total scan time

---
